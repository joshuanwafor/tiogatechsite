# Tioga Technologies Ltd

**IoT Infrastructure & Embedded Systems | Intelligent Renewable Energy Solutions**

Tioga Technologies Ltd is an IoT infrastructure and embedded systems company. We develop core infrastructure around IoT, embedded systems, and intelligent renewable energy solutions. Tioga is at the forefront of cutting-edge next-generation renewable energy management platforms designed to power Africa's clean energy transition.

## About Us

Tioga Technologies specializes in:

- **IoT Infrastructure**: Building robust, scalable IoT infrastructure solutions
- **Embedded Systems**: Developing custom embedded systems for various applications
- **Renewable Energy Management**: Creating intelligent platforms for renewable energy management
- **Clean Energy Transition**: Powering Africa's transition to clean energy through innovative technology

## Technology Stack

This website is built with:

- **Next.js 16** - React framework for production
- **React 19** - UI library
- **TypeScript** - Type-safe JavaScript
- **Tailwind CSS v4** - Utility-first CSS framework

## Getting Started

### Installation

```bash
npm install
# or
yarn install
# or
pnpm install
```

### Development

Run the development server:

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
```

Open [http://localhost:3000](http://localhost:3000) to view the website.

### Build

Build for production:

```bash
npm run build
```

Start production server:

```bash
npm start
```
